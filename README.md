# 🌿 Pepper Disease Detection

## 📌 Overview

This project focuses on detecting diseases in pepper leaves using deep learning techniques.
The entire implementation is provided in a Jupyter Notebook for easy understanding, experimentation, and reproducibility.

---

## 📓 Notebook

* `notebooks/pepper_disease_detection.ipynb`

The notebook includes:

* Data preprocessing
* Image enhancement using CLAHE
* Model building using CNN with CBAM
* Training and evaluation

---

## 📊 Results

* Successfully trained a model for pepper leaf disease classification
* Includes performance visualization (accuracy, loss graphs)
* Demonstrates prediction on sample images

---

## 📁 Dataset

* Dataset: PlantVillage (Pepper Leaf Dataset)

⚠️ Note: Dataset is not included in this repository due to size limitations.
You can download it from Kaggle or other public sources and place it in a `data/` folder.
